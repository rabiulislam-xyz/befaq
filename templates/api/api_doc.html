API URLs:

www.befaq.ml/api/result/ for single result

www.befaq.ml/api/result/top/ for top results list

Retrieve single result with Get request:

to retrieve a student result make a get request to www.befaq.ml/api/result/<ID>/
where ID is student's registration ID

example: go to www.befaq.ml/api/result/2/ for retrieve result of student whose registration ID is 2.

retrieve top results list:

Go to www.befaq.ml/api/result/top/ for top results.

if you want to filter top results on Madrasa, Marhala or Registrations year then parse them to get request parameters.

example: to retrieve Top result list of students whom Registration Year is 2018, Madrasa is: Jamia Arabia Imdadu Uloom, Marhala is: Takmil.
Go to These URL: www.befaq.ml/api/result/top/?madrasa=Jamia+Arabia+Imdadul+Uloom&marhala=takmil&reg_year=2018

You Can also submit Madrasa's Registraton ID instead Madrasa's Name
